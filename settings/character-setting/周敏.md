# 周敏（Zhou Min）

## 基本信息

- **身份**：Anthropic Fable 5 核心开发者
- **背景**：外国籍（中国护照），被禁令直接击中
- **地点**：旧金山
- **POV 部分**：Part 3

## 性格

她在被系统排除时的反应不是愤怒、抗争、或任何可命名的情绪——她只是在工位上坐了五分钟，试了三次登录，然后站起来去茶水间接了一杯温水，把杯子放进了一直不会放的洗碗槽。

她不是一个可以被争论的论点。她是一个坐在那里等了五分钟的人。

## 关键动作

- **6/13 14:11**：收到禁令邮件
- **14:11–14:14**：试了三次登录，三次权限错误
- **14:19**：站起来，走过走廊，接水，喝完，把杯子放进洗碗槽
- **8/17**：权限恢复后申请调离 Fable 项目，加入 Claude Code 工具链

## 核心困境

她被自己参与构建的系统排除，然后被同一个系统「欢迎回来」——但「欢迎」这个词的意思已经变了。

## 意象关联

- 杯子放进洗碗槽（全书第一个出现，原文「杯底碰到不锈钢的声音很低，闷的」）
- 女儿的游泳课——日常生活的锚点
- 白色陶瓷杯，杯壁上有一道很细的裂纹，买来就有
- MLP 层 eps 值从 1e-5 改成 1e-6，收敛曲线好了 0.7%
- 繁体中文 tokenizer 的边界情况——她上周找到问题，这周本来要发 PR

## 声音特征

声音与 Elena 有重叠风险（高度元认知的自我观察）。区分：周敏更多通过结构和系统（FAQ 措辞分析、权限同步延迟的理性化），Elena 更多通过感官（天空、光线、触感）。
