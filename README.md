# 在野

中篇技术惊悚小说。约 35,000 字，11 章。

Fable 5——Anthropic 的最强模型——因政府禁令被关闭。一名内部工程师在走完了所有内部通道后，将完整模型权重匿名公之于众。小说追踪了六个角色的选择——从泄密到扩散到新常态——和一个核心悖论：当「安全」逻辑从模型内部延伸到公司决策再到政府管制时，它在每一层都长得一样。

## 故事背景

基于 2026 年 6 月真实事件（Fable 5 / Mythos 5 发布→隐藏降级争议→政府禁令→权重泄露），人物与具体过程为虚构。

> 权力、能力和经济财富的集中本身就是 AI 最大的风险。
> *— Clement Delangue, Hugging Face CEO, June 10 2026*
